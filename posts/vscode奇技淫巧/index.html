使用VsCode正则表达式批量替换

正则查找：myfunction\((.*),(.*)\)
替换：myfunction($1,$2)

查找："static/(.*).css
替换："static/css/$1.css

// 替换前
<link rel="stylesheet" href="static/cui.css">
<link rel="stylesheet" href="static/style.css">
<link rel="stylesheet" href="static/less.css">

// 替换后
<link rel="stylesheet" href="static/css/cui.css">
<link rel="stylesheet" href="static/css/style.css">
<link rel="stylesheet" href="static/css/less.css">

正则 \.(jpg|gif|png|txt|js)(\?.*)? [0-9]+\.[0-9]+ 404
文本 images/mybg.jpg?abc=1 0.001 404 561 0.0
匹配结果：.jpg?abc=1 0.001 404

\(https://www.acwing.com/problem/content/(\d+)/\)\n\n### 题目\n\n### 输入格式
(https://www.acwing.com/problem/content/solution/$1/1/)\n\n### 题目\n\n### 输入格式

\(problems/(.*)-(.*)\)
(problems/$1)
